# Template de roadmap d’apprentissage / projet

## Structure standard
1. **Objectif** — résultat concret mesurable
2. **Prérequis** — ce qu’il faut déjà savoir
3. **Compétences cibles** — liste priorisée
4. **Stack** — outils choisis + pourquoi
5. **Étapes** — progression du simple au complexe
6. **Projets jalons** — un livrable par étape
7. **Difficultés** — pièges connus
8. **Durée estimée** — réaliste (avec marge)
9. **Ressources** — docs officielles d’abord

## Exemple jalons backend Python
- Étape 1 : Python + HTTP + JSON
- Étape 2 : API FastAPI CRUD
- Étape 3 : PostgreSQL + ORM
- Étape 4 : Auth JWT
- Étape 5 : Tests + Docker
- Étape 6 : Déploiement minimal

## Conseil mentor
Une roadmap sans projets concrets reste théorique. Chaque étape doit produire quelque chose que l’on peut démontrer.
